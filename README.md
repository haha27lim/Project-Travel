## Travel Planner Application

This is a full-stack web application designed to help users plan, organise, and manage their travel itineraries with AI-powered assistance. This application provides an intuitive interface for creating custom travel plans, managing trips, saving favorite places, and getting personalised recommendations through an intelligent AI assistant.

### Features
*   **Trip Management:** Create and manage trips. Share your travel itineraries with friends and family. Organise activities within your trips 
*   **Location & Mapping:** Google Maps integration for visualising destinations. Add, edit, and organise saved places with notes
*   **AI-Powered Suggested Itinerary:** Generate personalised travel itineraries using Google Vertex AI (Gemini). utomatically create detailed travel plans based on preferences
*   **User Management:** Manage personal information and preferences. Comprehensive admin panel for user and trip management
*   **Chatbot:** travel assistant for recommendations and help


### Tech Stack

*   **Backend:** Spring Boot (Java), Spring Security, Spring Data JPA
*   **Frontend:** React, Typescript, Redux Toolkit, Vite, shadcn/ui
*   **Database:** MySQL(Production), H2 (In-Memory for development/testing)
